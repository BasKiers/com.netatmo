# NetAtmo for Homey

This app supports multiple NetAtmo devices on Homey. Support includes:

- NetAtmo Weather Station + modules
    - Read sensors
        * Temperature
        * Humidity
        * CO2
        * Pressure
        * Noise
        * Rain
        * Humidity
        * WindStrength
        * WindAngle
        * GustStrength
        * GustAngle
- NetAtmo Thermostat
    - Read sensors
        * Temperature
        * Target Temperature
    - Set capability
        * Target Temperature
        * Active Program
        * Active Mode
